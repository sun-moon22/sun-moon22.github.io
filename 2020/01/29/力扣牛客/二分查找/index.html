二分查找

概念

在有序集合查找target过程

时间复杂度：logn

2要素：

  • Target目标值
  • 查找区间（left，rigth索引）

模板

例题

69. x 的平方根 - 力扣（LeetCode） (leetcode-cn.com)

  • 在1-x的区间上找到一个数mid，使得x/mid == mid即可；以二分查找的方式 找mid
  • 重点：二分查找的特点，若没有找到target，right指针一定指向最接近target

33. 搜索旋转排序数组 - 力扣（LeetCode） (leetcode-cn.com)

通过比较target和nums[0],确定目标在左半部分还是右半部分。

例如 target = 5, 目标值在左半段，因此在 [4, 5, 6, 7, inf, inf, inf] 这个有序数组里找就行了；
例如 target = 1, 目标值在右半段，因此在 [-inf, -inf, -inf, -inf, 0, 1, 2] 这个有序数组里找就行了。
如此，我们又双叒叕将「旋转数组中找目标值」 转化成了 「有序数组中找目标值」
